# このリポジトリは？
Kohchaさんが適当に作ったツールを公開してるVPMのカタログリポジトリです。

# ツールを追加したい場合
1. [ここ](https://kohchaaa.github.io/vpm-repos/) にアクセス
2. `Add to VCC`を押してVCC、ALCOMに追加
    - `https://Kohchaaa.github.io/vpm-repos/vpm.json`のリンクをVCC、ALCOMに貼り付けても可

# ツールについて
### WindowLocker
`Ctrl + L`でInspector、Projectウィンドウのロック状態を切り替えできるショートカットを追加
ショートカットはUnityの設定から変更可能
